# About STGR Browser

**Version:** 1.0.0  
**Build ID:** 20260701  
**Platform:** Gecko (Firefox-based)

STGR Browser is a lightweight, privacy-first browser built for performance and gaming. It is based on Mozilla's Gecko rendering engine and maintains full web standards compatibility.

## Key Features

- **Privacy First**: No telemetry, no tracking, no sponsored content
- **Gaming Mode**: Automatic performance optimization while gaming
- **Memory Optimized**: Aggressive memory management for low RAM usage
- **GPU Accelerated**: Full hardware acceleration with WebRender
- **Minimal UI**: Clean, modern interface with no clutter
- **Extension Compatible**: Works with thousands of Firefox extensions
